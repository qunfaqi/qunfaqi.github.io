## 如何替换

为了获得最佳效果，请按照以下建议替换这些占位图片：

- 所有图片应使用 PNG 或 JPG 格式
- 推荐图片尺寸为 800x600 像素或保持 4:3 的宽高比
- 确保图片清晰，质量高，能够展示软件的专业性
